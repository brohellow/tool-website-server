# 工具乐园

一个创意个性的多功能在线工具集合网站。

## 访问地址

http://20111108.xyz

## 功能特色

- 🛠️ **实用工具**：JSON 格式化、颜色选择器、二维码生成器等多种在线工具
- 💬 **实时聊天**：聊天室功能支持实时消息交流
- 🎮 **三国杀游戏**：多人在线卡牌游戏，支持武将选择
- ❤️ **用户系统**：注册登录、收藏工具、发表评论

## 技术栈

- **前端**: React 18 + TypeScript + Tailwind CSS 3 + Vite 5
- **后端**: Node.js + Express + Socket.IO
- **数据库**: 内存数据库 (memoryDb)
- **认证**: JWT
